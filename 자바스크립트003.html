제어문 (조건문, 반복문)

제어문이란? 프로그램에서 필요한 결과값을 도출하기 위해 실행문의 순서를 제어하거나 반복시키는 문장

  • 조건문 - if문 -> (조건형식){참일경우}else{거짓일경우}
  • 반목문
    • while문 - 변수설정을 let i=0 while (i < 8) { 실행문 ; 증강식;}
    • for문 - for(변수설정 let i=0; i < 8; 증강식){ 실행문}
조건문예제-질의응답형식
조건문예제-평균값내기
반복문예제 - 숫자1부터 10가지 출력하기(while문작성)
반복문예제 - 구구단 2단작성
