요소선택

Dom(document object model)의html 요소를 선택, 생성, 삭제, 등을 활용한다.

직접선택메서드

메서드이름 설명
getElementByID('') HTML요소중 ID 속성인 요소만 부름
getElementsClaccName('') HTML 요소중 Class속성인 요소를 모두 부름
getElementsTagName('') HTML 요소중 태그의 요소를 모두 부름
querySelector() 선택된 요소중 첫번째 요소만 선택 / 태그를 제외하고 틀래스 / 아이디를 부를때는 CSS와 동일하게 부름
querySelectorAll() getElementsTagName과 같은 성질 단, 태그네임외에 다른 중복요소 및 클래스도 반영가능

Event

선택자 혹은 도큐먼트상에서 행동을 가했을때 발생되는 현상

이벤트 설명
cilck 마우스를 클랙 했을떄 이벤트가 발생된다.
mouseover 마우스가 오버했을때 이벤트가 발생된다.
mousemove 마우스가 움직였을때 이벤트가 발생된다.
mouseenter 마우스가 들어갔을때 이벤트가 발생한다.
mouseleave 마우스가 떠났을때 이벤트가 발생한다.
submit submit을 눌렀을때 이벤트가 발생한다.
load 페이지를 열었을떄 이벤트가 발생한다.
resize 사이즈가 변경되었을때 이벤트가 발생한다.
scroll 스크롤바가 움직였을때 이벤트가 발생한다.
