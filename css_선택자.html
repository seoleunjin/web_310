선택자의 종류

html파일 css를 통해 꾸밀수 있고 꾸미기 위해서는 지정된 선택을 해야한다.

작성법

선택자 { 속성명: 속성값; }

선택자종류 작성방법 설명
전체선택자 * HTML 내부에 모든 태그를 불러서 결정한다.
태그선택자 태그명 HTML 내부에 불려진 모든 태그를 설정
아이디선택자 #아이디이름 속성값 아이디에 이름을 불러서 특정 아이디만 설정
클래스선택자 .클래스명 속성값 class의 이름을 불러서 특정 클래스만 설정
ID와 CLASS의 차이점:
id는 한페이지에 한번만 사용 중복사용이 안됨 한번 사용한 아이디 이름은 두번 사용안됨
클래스는 중복적용이 가능, 한 클래스에 여러 클래스명 사용가능
클래스명 한칸 띄어쓰기로 여러 클래스명 부여가능
중복선택자 , 여러개의 선택자를 나열해서 사용할수 있다.
중복선택자 붙여쓰기(태그.클래스명) 지정된 태그중 해당 클래스 이름만 부름
후손선택자 선택자 선택자 부모자 내부애 있는 요소를 모두 부른다.
자손선택자 부모자>자손자 다이렉트 자손자에 한에서 설정가능
