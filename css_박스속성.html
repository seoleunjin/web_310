박스속성

웹페이지 레이아웃 구성시 가장 중요

  • magin - 여백
  • padding - 영역
  • wide , height - 크기

작성법

  • 전체적용 - 50px 하나의 숫자로 4군데 모두적용
  • 각자적용 - 위쪽 오른쪽 아래쪽 왼쪽 / 모두 단위와 함께 작성 중간에 쉼표 쓰지 않음
    위아래 좌우 / 두번 두개 숫자로 첫번째 언급 숫자- 위와 아래 같이적용, 두번쨰숫자 - 좌우 같이적용 20px 30px
  • 부분적용
    • magin-left: 왼쪽여백
    • magin-right: 오른쪽여백
    • magin-top: 위쪽여백
    • magin-bottom: 아래쪽여백
    • padding-left : 왼쪽영역
    • padding-right : 오른쪽영역
    • padding-top : 위쪽영역
    • padding-bottom : 아래쪽영역

width : 가로 크기값
width+(padding+border)*2

height : 세로크기값
height+(padding+border)*2

border

박스의 테두리를 말하며 , 가로, 세로 값에 영행을 준다.
색상, 굵기 , 모양을 한번에 모두 표기시 보인다.

작성법
border : black 2px solid (순서와 상관없음)
4면 한번에 나오거나 지정할 수 있음 norder-left,right,top,bottom

선의종류
dashed(긴점선), dotted(짧은점선), double(두줄선), 액자선(groove, rifge, inset, outset), none(없음)

배경

배경색, 배경이미지, 배경반복, 배경위치, 배경고정 등을 설정한다.

  • background-color: 색깡이름, RBG, RBGA로 표기/ transperans(투명색)
  • background-image: url("위치주소기재")
  • background-repeat: (가로, 세로 모두 반복), repeat-x(가로반복), repreat-y(세로반복), no-repeat(반복없음)
  • background-position: 가로위치, 세로위치, 생략시 기본 값 성절 (left, top)
    숫자로 지정시 왼쪽상단 기준에서 시작됨 가로숫자,세로숫자
    위치명으로도 기재가능
  • background-attachmant : 배경이미지를 고정시킬지 스크롤 할 지 결정
    scroll, fixed
  • background-size
    • 100% - 화면을 비율 상관없이 꽉채움(찌그러짐)
    • contain - 배경이미지를 잘리는 부분 관계 없이 이미지 우선으로 보임(비율조정)
    • cover - 배경 이미지와 상관없이 화면을 꽉 채움
