입력양식태그란?

입력양식은 사용자에게 정보를 입력 받는 요소 예를들어/로그인/회원가입 겁색 등을 들수있다.

form 태그 내부에 입력양식태그들을 넣어서 만든다.
form 태그는 전달받는 곳에 형식을 기재해야 한다.

  • method="get" - 직접전달 (당근마켓)
  • method="post" - 비밀전달 (본인외 개봉금지)
from태그 외부에 입력양식 태그들은 기재할 수 있으나 전송할수는 없다.

입력양식 종류

태그명 속성명(type)="" 설명
form 특성 보이지 않음 / block 요소 입력양식의 시작과 끝
input text 글자입력양식
password 비밀번호양식설정
submit 제출버튼(확인)
reset 초기화버튼
file 내컴퓨터에 있는 파일입력 생성
search 검색박스로 텍스트 입력시 X버튼이 나타남
button 버튼생성버튼
checkbox 사각형의 선택박스 (다중선택)
radio radio버튼생성 (단일선택)
같은이름이라은 표식을 속성으로 해줌 name="본인이름"
email 입력양싱에 맞게 작성하지 않을 경우 오류발생
number 숫자입력방식 - 다양한 속성이 함께 작성가능
color 컬러를 선택할수 있는 기능
range 슬라이드바 - 숫자범위로 나타남, 다양한 속성값
datetime, month, week, date, time 국제표준시간(datetime-local로 인식가능), 연과월만표시, 연과주만표시, 연월일표시, 시간표시
required 필수작성표시
value"눈에보이는글씨" 외부에 보이는 글씨변경
placeholder="도움글" 입력양식에 힌트가 되는 글을 보여줌, 내용을 쓰기 시작하면 지워짐
autoforcus="autoforcus" 입력필드에 커서가 자동으로 생성 - 내용작성 유도
min="최소값" max="최대값" step="건너뛰기" 수 가 들어가야 하는 모든 속성에 사용가능
textarea csol="가로크기값(글씨가 들어가는 최소값)" rows="세로값 줄바꿈의 갯수" 여러행의 글자입력양식 텍스트와 크기 값이 같지않음
select multiple="multiple" 선택양식생성
option 옵션생성
optgroup label="그룹명" 옵션선택의 그룹지정으로 선택되지 않음
fieldset text그룹양식 자동으로 선으로 묶어줌
legend fieldset의 타이틀 선의 내부에 작성됨
label 태그 라는 정의태그로 폼태그 내부에 결합 하고자 하는 속성과 함께 사용된다.(인라인)
